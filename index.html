CPAN Meta DB

APIs

/package/Package::Name
Returns the latest distribution that contains the package and its version string ('undef' is a valid version string) in YAML format. Returns 404 status code if the package is not found.
This is yet another CPAN Meta DB, created by Tatsuhiko Miyagawa for cpanminus.
Powered by Google AppEngine and the source code is available on github.
